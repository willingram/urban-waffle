Zero to IAA in Sixty Minutes

Overview

  • Install the AWS Amplify CLI
  • Scaffold an app using Create React App 2
  • Add and configure Analytics, Hosting, Auth and an API
  • Analytics: Pinpoint (and optionally Kinesis)
  • Hosting: S3 (and optionally Cloudfront)
  • Auth: Cognito
  • API: AppSync, Lambda and DynamoDB
